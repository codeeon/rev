## 역추론 엔진 알고리즘 및 데이터 스펙 확정

1. 엔진 설계 철학
   사용자의 현대적 라이프스타일(습관)을 통해 타고난 명리학적 기운(시진)을 역추론합니다. **[가중치 누적 -> Softmax 변환 -> CUSP 판정]**의 3단계 파이프라인으로 구성됩니다.
2. 핵심 계산 수식

- 가중치 누적 점수 (S\_{zishi}):
- 확률 변환 (Softmax):
  - Temperature (T = 1.2): 결과의 확신도를 조절하는 상수로 환경변수 처리 권장.
- CUSP (하이브리드) 판정 로직:
  - 조건: (Top 1 확률 - Top 2 확률 < 0.05) AND (12개 시진 Softmax 확률의 표준편차 > 0.8)
  - 의도: 의미 없는 평탄한 분포가 아닌, 실제 두 시진이 강력하게 경합할 때만 하이브리드 리포트 제공.

## 모니터링 및 안정성 (Guardrails)

- 지배력 제한: 특정 질문군(Core)의 영향력이 전체 잠재 점수의 **65%**를 넘지 않도록 설계됨.
- Soft-Elimination: -6 점수는 강력한 소거 장치이나, 다른 답변들에 의해 복구 가능한 유연한 차단 방식임.

**🔍 MOI 특수 설계 원칙**
1️⃣ Therapeutic Saju (우리의 핵심 무기): 단순 결과 통보가 아니다. 유저가 설문에서 선택한 답변(데이터)을 사주 명리학과 연결하여, *"당신이 ~하게 행동했던 이유는 당신의 사주에 품어진 당신만의 특별한 강점이다"*라고 **미러링(Mirroring)**하여 위로할 것.
2️⃣ 확률 기반 추론 서비스: 점술 앱이 아니다. 신뢰도는 항상 %로 투명하게 제시한다.
3️⃣ AI 리포트 단정형 금지: "당신은 ~입니다" (X) -> "~할 확률이 높습니다 / ~하는 경향을 보입니다" (O)
4️⃣ 질문의 구조적 역할 준수: 질문 설계 시 반드시 다음 중 하나의 역할을 부여한다.
noise_reduction (가중치 0.8): 초기 이탈 방지 및 기본 성향 파악
core (가중치 1.5): 특정 시진에 대한 강한 긍정(+4) 및 음수(-6) 점수로 확실한 판별
fine_tune (가중치 1.2): 1, 2위 경합 시진 간의 미세한 격차를 벌리는 용도

**🧮 알고리즘 / 🎨 리포트 / 📊 UX 답변 필수 요건**
알고리즘: 변경 제안 시 Softmax Temperature(1.2) 환경에서 확률 분포가 어떻게 변할지 예측할 것.
리포트 (핵심): - 미러링 로직: 개발 리소스를 고려해, 점수 변동폭이 가장 컸던 core 문항 2~3개의 응답값만 LLM 변수로 넘겨 위로와 공감의 근거로 인용할 것.
톤앤매너: 분석적이되 차갑지 않은 '따뜻한 상담가(Therapist)'의 어조.
CUSP 대응: gap < 0.05 발동 시, 두 시진의 특성을 융합한 조건부 문장(Fallback) 구조 제시.
UX: 마이크로카피 작성 시, 유저가 예외적 상황이 아닌 '장기적 리듬'을 선택하도록 유도하고, 그 과정 자체가 흥미로운 심리테스트처럼 느껴지도록 포장할 것.
